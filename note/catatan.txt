# **Areas of Self-Directedness in Learning (Kemandirian dalam Belajar)**

**Kunci Jawaban:**
5 = Selalu
4 = Sering
3 = Kadang-kadang
2 = Jarang
1 = Tidak Pernah

---

## **1. Awareness (Kesadaran Diri dalam Belajar)**

1.1 Saya tahu apa yang perlu saya pelajari.
1.2 Saya bisa memilih cara belajar yang paling cocok untuk saya.
1.3 Saya melihat guru sebagai pembimbing belajar, bukan hanya pemberi materi.
1.4 Saya mencari tahu berbagai sumber belajar (buku, internet, video, dll).
1.5 Saya bertanggung jawab atas belajar saya sendiri.
1.6 Saya tahu bagian mana dari pelajaran yang masih kurang saya pahami.
1.7 Saya bisa memotivasi diri sendiri untuk belajar.
1.8 Saya bisa merencanakan dan menentukan target belajar.
1.9 Saya beristirahat sejenak saat belajar lama agar tetap fokus.
1.10 Saya bisa mengatur waktu belajar agar tidak bentrok dengan kegiatan lain.
1.11 Saya menghubungkan pengalaman saya dengan pelajaran baru.
1.12 Saya merasa tetap belajar walaupun tidak selalu dijelaskan guru.

---

## **2. Learning Strategies (Strategi Belajar)**

2.1 Saya aktif ikut diskusi kelompok.
2.2 Saya merasa belajar dengan teman (saling mengajar) itu membantu.
2.3 Saya merasa bermain peran/simulasi membantu memahami materi sulit.
2.4 Saya lebih paham saat belajar interaktif daripada hanya mendengarkan ceramah.
2.5 Saya merasa simulasi/praktik membantu saya belajar.
2.6 Saya merasa belajar dari studi kasus itu bermanfaat.
2.7 Saya punya dorongan dari dalam diri untuk terus berkembang.
2.8 Saya melihat masalah sebagai tantangan untuk belajar.
2.9 Saya membuat jadwal belajar yang teratur.
2.10 Saya merasa peta konsep/rangkuman membantu belajar.
2.11 Saya memanfaatkan teknologi (HP, laptop, aplikasi) untuk belajar.
2.12 Saya bisa menentukan cara belajar saya sendiri.

---

## **3. Learning Activities (Kegiatan Belajar)**

3.1 Saya mengulang dan mempelajari kembali materi.
3.2 Saya bisa menemukan poin penting saat membaca.
3.3 Saya membuat rangkuman/peta konsep agar lebih paham.
3.4 Saya bisa menggunakan teknologi untuk belajar.
3.5 Saya lebih fokus saat membaca materi sulit.
3.6 Saya membuat catatan atau rangkuman dari pelajaran.
3.7 Saya suka mencari informasi di luar materi sekolah.
3.8 Saya menghubungkan teori dengan praktik nyata.
3.9 Saya bertanya saat ada yang kurang jelas.
3.10 Saya memikirkan kembali dan menilai ide/informasi baru.
3.11 Saya terbuka dengan pendapat orang lain.
3.12 Saya beristirahat di sela-sela belajar.

---

## **4. Evaluation (Evaluasi Diri)**

4.1 Saya menilai hasil belajar saya sendiri sebelum dinilai guru.
4.2 Saya tahu bagian mana yang perlu saya tingkatkan.
4.3 Saya memantau perkembangan belajar saya.
4.4 Saya tahu kelebihan dan kekurangan saya dalam belajar.
4.5 Saya senang jika teman memberi masukan pada tugas saya.
4.6 Saya belajar dari keberhasilan dan kegagalan.
4.7 Saya menerima kritik untuk memperbaiki diri.
4.8 Saya mengecek apakah target belajar sudah tercapai.
4.9 Saya meninjau kembali catatan/portofolio belajar.
4.10 Saya merefleksikan kegiatan belajar saya.
4.11 Saya suka tantangan dalam belajar.
4.12 Saya termotivasi melihat keberhasilan orang lain.

---

## **5. Interpersonal Skills (Keterampilan Sosial dalam Belajar)**

5.1 Saya tertarik mempelajari budaya dan bahasa lain.
5.2 Saya tahu peran saya saat kerja kelompok.
5.3 Saya belajar dari berinteraksi dengan orang lain.
5.4 Saya memanfaatkan kesempatan untuk belajar hal baru.
5.5 Saya suka berbagi informasi dengan orang lain.
5.6 Saya menjaga hubungan baik dengan orang lain.
5.7 Saya mudah bekerja sama dalam kelompok.
5.8 Saya percaya diri berbicara di depan orang lain.
5.9 Saya memahami pentingnya kerja sama antar bidang.
5.10 Saya bisa menyampaikan ide dengan tulisan.
5.11 Saya berani menyampaikan pendapat.
5.12 Saya tertantang belajar di lingkungan yang beragam.
